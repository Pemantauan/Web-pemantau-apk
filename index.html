Red Flags Terdeteksi
  • 💰
    Return tidak realistis
    Klaim return hingga 375% dalam 40 hari. Tidak ada bisnis konten digital yang mampu menghasilkan ini secara konsisten.
  • 👥
    Sistem referral berlapis
    Komisi rekrutmen multi-level adalah ciri khas skema Ponzi/MLM ilegal.
  • 🔒
    Biaya aktivasi & upgrade
    Wajib bayar untuk akses fitur profit. Uang masuk = bahan bakar Ponzi.
  • 🌐
    Domain ganda
    shortpro.app & shortpro.co.id. Platform legit hanya punya satu domain utama.
  • 📋
    Tidak terdaftar OJK
    Menawarkan "investasi" tanpa izin OJK/Bappebti = aktivitas ilegal di Indonesia.
Prediksi Fase Collapse
Fase 1 — Honeymoon
Semua orang bisa withdraw lancar. Testimoni positif tersebar. Rekrutmen masif. (Mungkin sedang di sini)
Fase 2 — Tekanan
Withdraw mulai dipersulit. Syarat tambahan muncul (upgrade lagi, minimum WD naik, "verifikasi"). Server lambat.
Fase 3 — Collapse
Withdraw diblokir total. Admin tidak responsif. Aplikasi tiba-tiba down. Kantor kosong.
Preseden — VTube 2021
Pola identik: nonton video → reward → collapse setelah rekrutmen berhenti. Ribuan korban, uang hilang.
Tanda Bahaya Sekarang
  • Withdraw sudah dipersulit / ada syarat baru
  • Admin lambat/tidak respon di grup
  • Ada tawaran "upgrade paket" untuk bisa tarik uang
  • Server sering down atau lambat
  • Minimum withdraw tiba-tiba naik
Centang yang sudah terjadi. Makin banyak → makin dekat collapse.
